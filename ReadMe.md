 
# Baker Soft time tracker

A simple API for tracking work time on prokects

## Description

This API allows you to create multiple users and projects, you can then assign tasks to users

## Getting Started

### Dependencies

* Python 3.6 and above
* Refer to the requirements.txt file

### Installing

* pip install -r requirements.txt
* A running virtual environment is advised

### Executing program

* navigate to the project folder
* run 'python manage.py makemigrations' and 'python manage.py migrate'
* run 'python manage.py runserver'
